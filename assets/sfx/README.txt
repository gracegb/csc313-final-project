Drop real WAV files in this folder to override synthesized fallback sounds.

Expected file names:
- ui_move.wav
- ui_confirm.wav
- ui_ready.wav
- round_start.wav
- punch.wav
- kick.wav
- hit.wav
- ko.wav

Notes:
- Use short PCM WAV clips (mono or stereo both work).
- Recommended sample rates: 22050 Hz or 44100 Hz.
- If a file is missing, the game falls back to the built-in synthesized effect.
